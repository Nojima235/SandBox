【取扱説明書】契約ナビ

使い方

  1. ヒアリングシート の項目をすべて埋める → この条件で開始
  2. 次へ / 戻る でステップを移動
  3. コンサルツールからプランおよび購入方法などを設定（準備中）
  4. チェックリストを埋めながら進行

　　【注意】リセットは入力・チェック・進捗を初期化します
